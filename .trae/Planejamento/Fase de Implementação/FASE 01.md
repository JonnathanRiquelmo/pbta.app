# FASE 01 — Setup do Projeto

**Objetivo**
- Inicializar o projeto com React + TypeScript + Vite, configurar lint, testes e estrutura base de pastas.

**Tarefas**
- Criar projeto com Vite (`react-ts`) e configurar `vite.config.ts` com `base` adequado.
- Adicionar `eslint`, `vitest`, `react-testing-library` e scripts em `package.json`.
- Criar estrutura de diretórios conforme componentes, hooks, contexts, services e utils.

**Referências**
- `.trae/Planejamento/Descrição das Tecnologias.md:5-14`
- `.trae/Planejamento/Deploy e CI-CD.md:52-68` (scripts) e `.trae/Planejamento/Deploy e CI-CD.md:71-78` (base do Vite)
- `.trae/Planejamento/Estrutura de Componentes React.md:7-86`
- `.trae/Planejamento/Arquitetura do Sistema.md:30-41`

**Critérios de Aceite**
- Projeto roda com `npm run dev` e compila com `npm run build`.
- Estrutura de pastas criada conforme hierarquia definida.
- Lint e testes executam sem erros.
