# Descrição das Tecnologias — pbta.app

## 2. Descrição das Tecnologias

- Frontend: React@18 + TypeScript + Vite
- Estilização: Tailwind CSS@3 + CSS Modules
- Gerenciamento de Estado: React Context + useReducer
- Roteamento: React Router DOM@6
- Firebase: SDK v9 modular (Auth, Firestore, Storage)
- Markdown: marked@4 + DOMPurify
- Build Tool: Vite@4
- Testes: Vitest + React Testing Library
- PWA: Workbox@6 para offline
- CI/CD: GitHub Actions
